# 발표 데모 스크립트

## 1분 소개

K-Game은 오늘의 단어와 프롬프트 룸을 통해 AI 응답과 사용자 참여를 함께 보여주는 서비스입니다.  
프론트는 정적 자산으로 배포하고, API는 서버에서 인증과 비즈니스 로직을 담당하며, AI 작업은 Lambda로 분리했습니다.

## 시연 순서

1. 첫 화면에서 서비스 목적을 설명합니다.
2. 게스트 또는 소셜 로그인 흐름을 보여줍니다.
3. Daily Word에서 문제와 응답 흐름을 시연합니다.
4. 힌트와 판정 응답이 분리된 구조를 설명합니다.
5. 프로필과 관리 화면으로 이어지는 사용자 흐름을 보여줍니다.
6. 문서와 Terraform 구조를 열어 아키텍처를 설명합니다.

## 강조 포인트

- 프론트 / API / Lambda / DB의 책임 분리
- 세션 쿠키 기반 인증
- Terraform과 문서로 설명 가능한 배포 구조
- GitHub 공개를 고려한 저장소 정리
