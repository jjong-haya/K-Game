# 보안 체크리스트

## GitHub 공개 전

- [ ] 실제 `.env`가 커밋되지 않았는지 확인
- [ ] 로그, zip, build 산출물이 추적 대상에 없는지 확인
- [ ] 문서에 로컬 절대 경로가 없는지 확인
- [ ] 개인 연락처나 개인 식별 정보가 남아 있지 않은지 확인

## 애플리케이션

- [ ] 세션 쿠키가 `httpOnly` 기준인지 확인
- [ ] `helmet`과 CORS 설정이 적용되었는지 확인
- [ ] 실패 응답이 HTTP 200으로 숨겨지지 않는지 확인
- [ ] 입력 검증이 누락되지 않았는지 확인

## AWS

- [ ] S3 공개 범위가 최소화되었는지 확인
- [ ] EC2와 RDS가 public exposure 없이 연결되는지 확인
- [ ] IAM 권한이 최소 권한인지 확인
- [ ] 비밀값이 Secrets Manager 또는 SSM 기준인지 확인

## 운영

- [ ] CloudWatch 로그와 알람이 있는지 확인
- [ ] schema 적용과 개발 seed 적용이 분리되었는지 확인
- [ ] 롤백 절차가 문서와 스크립트에 함께 있는지 확인
