# 변경 기록

이 문서는 저장소의 큰 구조 변경과 사용자 영향이 있는 작업만 기록합니다.

## [Unreleased]

### 추가 및 개선

- `client`, `server`, Lambda, `deploy` 중심 구조를 `apps`, `services`, `infra`, `docs`, `.github` 중심 monorepo로 재정리했습니다.
- 프론트엔드를 CRA에서 Vite 기반 구조로 전환했습니다.
- 프론트 인증 흐름을 세션 쿠키 기반으로 정리했습니다.
- 백엔드 API를 라우트, 서비스, 저장소, 통합 계층으로 분리했습니다.
- Lambda 응답 계약을 통일하고, 실패 응답의 HTTP 200 고정 패턴을 제거했습니다.
- 배포 스크립트와 Terraform 문서를 공개 저장소 기준으로 다시 정리했습니다.
- GitHub 공개용 메타 문서와 품질 파일을 추가했습니다.

### 보안 및 운영

- `.env`, 로그, 빌드 산출물, 실험성 파일을 Git 추적 대상에서 제외하는 기준을 강화했습니다.
- 보안 정책, 기여 가이드, 릴리즈 점검 흐름을 문서화했습니다.
- Dependabot, CI, CODEOWNERS를 추가해 유지보수 흐름을 정리했습니다.

## [0.1.0] - 2026-04-11

- AWS 기반 웹서비스용 기본 아키텍처 구성
- 프론트엔드, 백엔드, Lambda, 인프라, 문서 체계의 기본 골격 마련
